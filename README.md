## ΏθΝθ What is DevForge AI?

**DevForge AI** is a powerful, step-by-step **Technical Blueprint Generator** that transforms a raw software idea into a complete engineering specification in minutes. No more blank-page syndrome. No more missed requirements. Just describe your idea έΑΦ DevForge AI handles the rest.

> _"Think of it as having a senior software architect available 24/7, who never gets tired and never misses a detail."_

---

## έερ Features

| Feature | Description |
|---|---|
| Ώθπι **AI-Powered Blueprints** | Generates complete technical specs from a single idea |
| ΏθΥΜ **Requirements Analysis** | Auto-generates functional & non-functional requirements |
| ΏθΩΓΎ╕Π **Use Case Diagrams** | Visual actor-system interaction models |
| ΏθΦΕ **Sequence Diagrams** | Step-by-step interaction flows between components |
| ΏθΠΩΎ╕Π **Class Diagrams** | OOP structure with attributes, methods & relationships |
| ΏθΟρ **UI/UX Mockups** | Screen layouts with design system (colors, typography) |

| ΏθΩΕΎ╕Π **Database Schema** | Full table definitions with fields and relationships |
| ΏθΥΒ **Folder Structure** | Frontend & backend directory organization |
| ΏθΝΛ **Flow Diagrams** | System flow + user journey visualizations |

| ΏθΥν **Markdown Export** | Download your full blueprint as a `.md` file |

## ΏθγΑ Getting Started

### Prerequisites

Make sure you have the following installed:

```bash
node  >= 20.0.0
npm   >= 9.0.0
```

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/your-username/devforge-ai.git
cd devforge-ai/app
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm run dev
```

**4. Open in browser**

```
http://localhost:5173
```

> ΏθΟΚ That's it! No API keys, no environment setup. It works right out of the box.
