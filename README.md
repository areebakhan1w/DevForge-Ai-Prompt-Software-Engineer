## ЁЯМЯ What is DevForge AI?

**DevForge AI** is a powerful, step-by-step **Technical Blueprint Generator** that transforms a raw software idea into a complete engineering specification in minutes. No more blank-page syndrome. No more missed requirements. Just describe your idea тАФ DevForge AI handles the rest.

> _"Think of it as having a senior software architect available 24/7, who never gets tired and never misses a detail."_

## ЁЯЪА Getting Started

### Prerequisites

Make sure you have the following installed:

```bash
node  >= 20.0.0
npm   >= 9.0.0
```

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/areebakhan1w/Dev-Forge-AI-Prompt-Software-Engineer.git
cd Dev-Forge-AI-Prompt-Software-Engineer/app
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm run dev
```

**4. Open in browser**

```
http://localhost:5173
```

> ЁЯОЙ That's it! No API keys, no environment setup. It works right out of the box.
